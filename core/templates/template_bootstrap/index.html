Quero ser um desenvolvedor

Tecnologias abordadas

Este é o 1º módulo do curso, nele você aprender os fundamentos da Web, como ela funciona de fato e os principais conceitos que todo desenvolvedor web precisa ter na ponta da língua, como DNS, domínio, protocolos, servidores, hospedagens, sites e muito mais.

Este será o 2º módulo do curso, nele você aprenderá a principal linguagem para construção de softwares e sites para a web. Neste módulo veremos os principais elementos da HTML e da HTML5, tags para SEO, web semântica e muito mais.

No 3º módulo do curso você aprenderá o CSS3, com ele você poderá dar estilo, beleza, usabilidade e até comportamentos para suas páginas construídas com HTML5. Aprenderá os principais conceitos como seletores, box model, text efect e muito mais.

No 4º módulo você aprenderá a linguagem Java Script, uma veterana em crescente ascensão no mercado de desenvolvimento web. Com ela você poderá dar comportamentos, adicionar validações e com isso aumentar a usabilidade de suas páginas construídas com HTML5 e CSS3

No 5º módulo do curso você aprenderá o seu primeiro framework, o Bootstrap 3, o Bootstrap e o mais conhecido e consolidado framework CSS do mercado, usaremos o CSS para construir páginas bonitas e usuais sem ter que escrever muito CSS, isso irá aumentar e muito a sua produtividade.

No 6º módulo do curso, aprenderemos a cuidadar do nosso código fonte, criando versões dos nosso códigos, trabalhar em equipe no mesmo código, transferir o código para servidor, e muito mais, para isso aprenderemos a tão respeitada ferramenta de versionamento de código, o GIT.

No 7º módulo do curso aprenderemos a nossa segunda linguagem de programação (depois do Java Script) que o e Python. O Python e considerada uma das mais inovadoras e promissoras linguagens do mercado, apoiada por nada menos que a Google, NASA, Globo.com e muitas outras.

No 8º módulo do curso aprenderemos lógica de programação. Faremos uma série de exercicios e desafios usando Python e Java Script para desenvolvermos as técnicas de raciocínio computacional que é tão essencial para todo BOM desenvolvedor.

No 9º módulo do curso aprenderemos o nosso segundo framework, o Django. O Django e um poderoso framework de desenvolvimento web ágil baseado em Python, e é com ele que vamos construir o coracão dos nossos sites e softwares web.

No 10º módulo do curso aprenderemos a usar o banco de dados MySQL com o framework Django. Aprenderemos a instalar e configurar o Django para se comunicar com este poderoso banco de dados.

No 11º módulo do curso aprenderemos a integrar o Django com mais um poderoso banco de dados, o PostgreSQL. Da mesma forma que o MySQL, aprenderemos a instalar, configurar e integrar o Django com este concorrente de peso do MySQL.

No 12º módulo do curso aprenderemos a colocar os nossos sites e sistemas em um servidor para serem acessados do mundo inteiro e para isso aprenderemos o Heroku. O Heroku e um SaaS de alta qualidade que nos fornece um ambiente fácil e rápido para hospedarmos nossas aplicacoes Python.
